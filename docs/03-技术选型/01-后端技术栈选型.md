# 后端技术栈选型

## 1. 运行时选型

| 选项 | 选择 | 理由 |
|---|---|---|
| .NET 版本 | **.NET 10 LTS** | 2026年11月发布LTS，最长支持期，最新语言特性 |
| 开发语言 | **C# 14** | 伴随 .NET 10，支持集合表达式、半自动属性等 |
| 运行时 | **ASP.NET Core 10** | 高性能、跨平台、容器原生支持 |

### .NET 10 关键特性

- **C# 14**: 扩展 nameof、集合表达式、字段关键字
- **Native AOT**: 预编译为机器码，极速启动
- **性能优化**: ASP.NET Core 持续性能提升
- **OpenAPI 3.1**: 内置 OpenAPI 生成
- **Aspire**: 云原生应用编排

| 组件 | 选择 | 说明 |
|---|---|---|

## 3. ORM 选型

| 组件 | 选择 | 说明 |
|---|---|---|
| ORM | Entity Framework Core 10 | 官方ORM，成熟稳定 |

### EF Core 数据查询增强

```csharp
// 自研 SqlMap 查询条件构造器 (类比 JeeSite SqlMap)
var query = _context.Users
    .Where(u => u.UserName.Contains(request.UserName))  // LIKE 查询
    .WhereIf(!string.IsNullOrEmpty(request.UserType),
        u => u.UserType == request.UserType)            // 条件查询
    .Where(u => u.Status != "1")                        // 软删除过滤
    .OrderBy(u => u.CreateDate)                         // 排序
    .ToPageResult(request);                             // 分页扩展
```

## 5. 缓存选型

| 组件 | 选择 | 说明 |
|---|---|---|
| 本地缓存 | MemoryCache | 进程内 L1 缓存 |
| 分布式缓存 | Redis | 共享 L2 缓存 |
| 二级缓存 | FusionCache | L1+L2 + 回退 + 防崩溃 |
| Session | Redis 共享 | 集群部署 |
| 缓存管理 | 统一 CacheUtils | 界面临控 |

```csharp
// FusionCache 使用示例 (类比 J2Cache)
public class CacheService
{
    private readonly IFusionCache _cache;

    public async Task<T?> GetOrSetAsync<T>(string key, Func<Task<T>> factory)
    {
        return await _cache.GetOrSetAsync(key, factory,
            options => options
                .SetDurationSeconds(60)            // L1 过期
                .SetFailSafe(true, TimeSpan.FromHours(2))); // 防崩溃
    }
}
```
