## 1. 缓存中间件

### 1.1 FusionCache (二级缓存)

类比 JeeSite J2Cache (Caffeine + Redis):

```
┌──────────────────────────────────────────────────┐
│                  应用程序                           │
│  ┌──────────────────────────────────────────────┐ │
│  │          IFusionCache API                    │ │
│  └──────┬──────────────────────────────┬────────┘ │
│         │                              │           │
│  ┌──────▼──────┐              ┌───────▼────────┐  │
│  │   L1 Cache   │              │   L2 Cache      │  │
│  │  MemoryCache │              │    Redis         │  │
│  │  极速(本进程) │              │  共享(跨进程)    │  │
│  └─────────────┘              └────────────────┘  │
│         │                              │           │
│         └──────────────┬───────────────┘           │
│                        │                           │
│              ┌─────────▼──────────┐                │
│              │  FailSafe (防崩溃)  │                │
│              │  缓存故障降级策略    │                │
│              └────────────────────┘                │
└──────────────────────────────────────────────────┘
```

核心配置:

```csharp
builder.Services.AddFusionCache()
    .WithDefaultEntryOptions(options => options
        .SetDuration(TimeSpan.FromMinutes(5))
        .SetFailSafe(true, TimeSpan.FromHours(2))
        .SetFactoryTimeToLive(TimeSpan.FromSeconds(30)))
    .WithSerializer(new FusionCacheSystemTextJsonSerializer())
    .WithBackplane(new RedisBackplane(() => redisConnection))
    .RegisterForReconnectionNotifications();
```

### 1.2 缓存场景

| 场景 | 缓存Key | 过期策略 |
|---|---|---|
| 字典数据 | `dict:{type}` | 30分钟 + 防崩溃 |
| 用户权限 | `perm:{userId}` | 10分钟 |
| 系统配置 | `config:{key}` | 1小时 |
| 菜单树 | `menu:tree` | 30分钟 + 变更清理 |
| 用户Session | `session:{token}` | 按会话过期时间 |

## 2. 消息队列

### 2.1 RabbitMQ + MassTransit

```csharp
// 事件总线配置
builder.Services.AddMassTransit(x =>
{
    x.AddConsumer<UserCreatedEventConsumer>();

    x.UsingRabbitMq((context, cfg) =>
    {
        cfg.Host("localhost", "/", h => {
            h.Username("guest");
            h.Password("guest");
        });

        cfg.ReceiveEndpoint("user-created-queue", e =>
        {
            e.ConfigureConsumer<UserCreatedEventConsumer>(context);
        });
    });
});
```

### 2.2 事件列表

| 事件 | 触发时机 | 消费者 |
|---|---|---|
| `UserCreatedEvent` | 用户创建 | 发邮件、推送通知、初始化空间 |
| `UserLoggedInEvent` | 用户登录 | 记录日志、更新最后登录 |
| `OrderSubmittedEvent` | 订单提交 | 库存扣减、通知处理 |
| `FileUploadedEvent` | 文件上传 | 文件预览转换、病毒扫描 |

## 3. 工作流引擎

### 3.1 Elsa Workflows 3.x

类比 JeeSite 的 Flowable 工作流:

```csharp
// 注册 Elsa
builder.Services.AddElsa(elsa =>
{
    // 工作流管理 (定义、版本、实例)
    elsa.UseWorkflowManagement(management =>
    {
        management.UseEntityFrameworkCore(ef =>
        {
            ef.UseSqlServer(connectionString);
        });
    });

    // 工作流运行时
    elsa.UseWorkflowRuntime();

    // HTTP 活动
    elsa.UseHttp();

    // 定时活动
    elsa.UseScheduling();

    // REST API
    elsa.UseWorkflowsApi();
});
```

### 3.2 支持的工作流能力

| 特性 | 说明 |
|---|---|
| BPMN 2.0 可视化设计器 | Elsa Studio 拖拽设计 |
| 长/短运行工作流 | 支持持久化状态 |
| 退回/自由流/会签 | 中国式工作流特性 |
| 自定义活动 | C# 扩展 Activity |
| 版本管理 | 多版本并行 |

## 4. 后台任务

### 4.1 Hangfire

类比 JeeSite 的 Quartz 在线调度:

```csharp
// 注册 Hangfire
builder.Services.AddHangfire(config =>
    config.UseSqlServerStorage(connectionString));

builder.Services.AddHangfireServer();

// 在线调度配置
app.UseHangfireDashboard("/jobs");

// 周期性任务
RecurringJob.AddOrUpdate<ISomeService>(
    "data-cleanup",
    service => service.CleanupOldDataAsync(),
    Cron.Daily(3, 0)); // 每天凌晨3点
```

## 5. 可观测性

### 5.1 OpenTelemetry

```csharp
builder.Services.AddOpenTelemetry()
    .WithMetrics(metrics => metrics
        .AddAspNetCoreInstrumentation()
        .AddHttpClientInstrumentation()
        .AddPrometheusExporter())
    .WithTracing(tracing => tracing
        .AddAspNetCoreInstrumentation()
        .AddHttpClientInstrumentation()
        .AddSqlClientInstrumentation()
        .AddOtlpExporter());
```

### 5.2 监控指标

| 指标 | 来源 | 说明 |
|---|---|---|
| HTTP 请求量/延迟 | ASP.NET Core 指标 | API 监控 |
| SQL 执行统计 | EF Core 拦截器 | 慢查询 |
| 缓存命中率 | FusionCache | 缓存效率 |
| 消息队列深度 | MassTransit | 积压监控 |
| JVM/.NET GC | OpenTelemetry | 内存分析 |
