# 数据库与存储选型

## 1. 数据库选型

### 1.1 支持数据库

继承 JeeSite 5 的多数据库支持策略:

| 数据库 | 支持级别 | 备注 |
|---|---|---|---|
| SQL Server 2019+ | ✅ 主推 | ASP.NET Core 原生支持最佳 |
| PostgreSQL 15+ / GaussDB | ✅ 主推 | Npgsql 提供支持 |
| SQLite | ✅ 开发测试 | 本地快速验证 |
| 达梦 DM8 | ✅ 已完成 | DmProvider 反射调用 |
| 人大金仓 KingbaseES | ✅ 已完成 | Kdbndp 反射调用 |

### 1.2 数据库连接配置

```json
{
  "ConnectionStrings": {
    "DefaultConnection": "Server=localhost;Database=JeeSiteNET;Trusted_Connection=true;TrustServerCertificate=true"
  },
  "DatabaseProvider": "SqlServer"  // SqlServer | PostgreSql | MySql | Sqlite
}
```

### 1.3 多数据库支持实现

```csharp
public static class DbContextExtensions
{
    public static IServiceCollection AddJeeSiteDbContext<TContext>(
        this IServiceCollection services,
        IConfiguration configuration)
        where TContext : DbContext
    {
        var provider = configuration["DatabaseProvider"] ?? "SqlServer";
        var connectionString = configuration.GetConnectionString("DefaultConnection");

        services.AddDbContext<TContext>(options =>
        {
            switch (provider)
            {
                case "SqlServer":
                    options.UseSqlServer(connectionString, sql =>
                        sql.MigrationsAssembly("JeeSiteNET.Infrastructure.SqlServer"));
                    break;
                case "PostgreSql":
                    options.UseNpgsql(connectionString, npgsql =>
                        npgsql.MigrationsAssembly("JeeSiteNET.Infrastructure.PostgreSql"));
                    break;
                case "MySql":
                    options.UseMySql(connectionString, ServerVersion.AutoDetect(connectionString),
                        mysql => mysql.MigrationsAssembly("JeeSiteNET.Infrastructure.MySql"));
                    break;
                case "Sqlite":
                    options.UseSqlite(connectionString);
                    break;
            }
        });

        return services;
    }
}
```

## 2. 对象存储

### 2.1 存储方案

| 方案 | 说明 | 选择 |
|---|---|---|
| 本地磁盘 | 开发环境 | 默认 |
| MinIO | 自建对象存储 | 推荐私有部署 |
| 阿里云 OSS | 公有云 | 按需 |
| 腾讯云 COS | 公有云 | 按需 |
| S3 兼容存储 | 通用接口 | MinIO/华为云等 |

### 2.2 统一 OSS 接口

```csharp
public interface IFileStorage
{
    /// <summary>上传文件</summary>
    Task<string> UploadAsync(string bucket, string key, Stream content);

    /// <summary>下载文件</summary>
    Task<Stream> DownloadAsync(string bucket, string key);

    /// <summary>删除文件</summary>
    Task DeleteAsync(string bucket, string key);

    /// <summary>生成预签名 URL (临时访问)</summary>
    Task<string> GeneratePresignedUrlAsync(string bucket, string key, TimeSpan expiry);

    /// <summary>获取文件信息</summary>
    Task<FileInfo> GetFileInfoAsync(string bucket, string key);
}
```

### 2.3 MinIO 配置

```csharp
builder.Services.AddMinio(options =>
{
    options.Endpoint = "localhost:9000";
    options.AccessKey = "minioadmin";
    options.SecretKey = "minioadmin";
});

builder.Services.AddSingleton<IFileStorage, MinioFileStorage>();
```
