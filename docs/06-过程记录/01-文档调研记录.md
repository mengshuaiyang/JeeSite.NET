## 各页面关键信息摘要

### 1. 文档总入口 (/) 
- 文档分为7大章节: 快速了解、后端开发、Vue前端、经典前端、扩展功能、云服务、技术支持
- 共42个文档子节
- 2026年最新版本 v5.17.1

### 2. 架构特点 (/feature/)
- 微内核 + 插件架构
- @Table 注解: 类头定义Column，自动生成CRUD SQL
- 数据权限: 三级 (认证→功能→数据行)
- J2Cache: Caffeine L1 + Redis L2
- 树表辅助字段机制

### 3. 技术选型 (/technology/)
- 主流框架: Spring Boot / Shiro / MyBatis
- 前端双轨: Vue3(Vite) / Beetl(Bootstrap)
- 17+ 数据库支持
- 微服务: Spring Cloud / Nacos / Sentinel

### 4. 功能介绍 (/function/)
- 组织管理: 用户/机构/公司/岗位
- 权限管理: 角色/二级管理员/系统管理员/审计
- 系统设置: 菜单/模块/参数/字典/国际化
- 系统监控: 日志/缓存/服务器/作业/在线用户

### 5. 开发规范 (/standard/)
- UI 设计: 对齐/布局/色彩/导航
- 数据库: 命名/设计/使用规范
- 分层命名: Controller/Service/Dao/Entity

### 6. 目录结构 (/catalog/)
- common: 通用工具模块
- modules/core: 核心功能模块
- 代码生成模板: crud/crud_cloud/module

### 7. 树表设计 (/treetable/)
- 7个辅助字段: parent_code/codes/tree_sort/sorts/leaf/level/names
- TreeService 封装完整 CRUD
- 10个示例查询场景

## 调研结论

1. JeeSite 架构稳定成熟，设计理念适合 .NET 移植
2. 树表辅助字段和 @Table 注解体系是核心设计亮点
3. 模块化设计、数据权限、代码生成是三大核心能力
4. 前端双轨策略值得借鉴
