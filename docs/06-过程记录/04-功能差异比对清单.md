## 7. JeeSite.NET 独有优势

- **现代化架构**: .NET 10 + ASP.NET Core + EF Core + Vue 3 + Ant Design Vue
- **国产数据库原生支持**: 达梦 DM8 + 人大金仓 KingbaseES + GaussDB
- **国密算法**: SM2 (ECDSA) + SM3 (Hash) + SM4 (CBC) 完整支持
- **容器化部署**: Docker Compose 一键拉起全部服务 (含 healthcheck)
- **AI 原生集成**: DeepSeek/AI Tools Function Calling + 向量检索 RAG
- **实时通信**: SignalR 替代传统消息队列推送
- **文件预览**: Playwright PDF 转换 + 缩略图 + 分块上传
- **代码生成**: 多数据库 Introspect + 生成/下载 ZIP
- **工作流**: Elsa 3.x 可视化工作流引擎
- **全文搜索**: Elasticsearch 集成
