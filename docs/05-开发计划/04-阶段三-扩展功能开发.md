## 1. 第11-12周: 工作流引擎集成 (BPM)

| 任务 | 技术 | 说明 |
|---|---|---|
| Elsa Workflows 集成 | Elsa 3.x | 安装 NuGet 包 + 配置 |
| 自定义活动开发 | C# Activity | 审批、会签、退回 |
| 流程设计器集成 | Elsa Studio Blazor | 可视化拖拽 |
| 流程管理 | API 封装 | 部署/启动/挂起/版本 |
| 任务管理 | TaskService | 待办/已办/转办/委托 |
| 流程跟踪 | 流程图显示 | 流转状态可视化 |

**关键实现**:
```csharp
// 自定义审批活动
public class ApprovalActivity : CodeActivity
{
    public Input<string> Approver { get; set; } = default!;
    public Output<bool> Approved { get; set; } = default!;

    protected override void Execute(ActivityExecutionContext context)
    {
        // 创建审批任务
        var taskService = context.GetRequiredService<ITaskService>();
        taskService.CreateTaskAsync(new TaskModel
        {
            Assignee = context.Get(Approver),
            ActivityId = context.ActivityId,
        });
    }
}
```
