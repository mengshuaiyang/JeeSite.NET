# 代码生成机制设计

> 继承 JeeSite 5 的代码生成理念，实现在线建表 → 代码生成 → 部署运行的完整链路。

## 1. 设计目标

- 减少 70% 的重复 CRUD 代码编写
- 支持单表/主子表/树表三种模式
- 同时生成后端 (C#) 和 前端 (Vue) 代码
- 支持自定义模板扩展
- 生成的代码可直接部署运行

## 2. 生成流程

```
┌─────────────────────────────────────────────────────────────┐
│  1. 在线建表                                                   │
│  Web界面 → 配置表名/字段/类型/索引 → 保存到数据库(或直接建物理表)  │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  2. 选择生成模板                                               │
│  单表 / 主子表 / 树表 / 仅持久层 / 模块                        │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  3. 代码生成                                                   │
│  Razor 模板引擎 + 表元数据 → 生成文件流                        │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  4. 文件输出                                                   │
│  后端: Entity/Service/Controller/EF配置                       │
│  前端: List.vue / Form.vue / API接口                          │
│  菜单SQL: INSERT INTO Sys_Menu                                │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  5. 部署运行                                                   │
│  复制文件 → 添加数据库迁移 → 运行菜单SQL → 重启验证             │
└─────────────────────────────────────────────────────────────┘
```

## 3. 模板引擎设计

使用 Razor 模板引擎 (`.cshtml` / `.tpl`):

```razor
@* Entity.cs.tpl - 实体生成模板 *@
@using JeeSiteNET.Core.CodeGen
@model TableMetaData

// <auto-generated />
using JeeSiteNET.Core;
using JeeSiteNET.Modules.@Model.ModuleName.Domain;

namespace @(Model.Namespace).Domain.Entities;

/// <summary>@Model.TableComment</summary>
public class @Model.EntityName : DataEntity<@Model.EntityName>
{
    @foreach (var col in Model.Columns.Where(c => !c.IsBaseField))
    {
    <text>/// <summary>@col.Comment</summary>
    public @col.CSharpType @col.PropertyName { get; set; }
    </text>
    }
}
```

```razor
@* Service.cs.tpl - 应用服务生成模板 *@
@model TableMetaData

public class @(Model.EntityName)Service : CrudService<@(Model.EntityName), @(Model.EntityName)Dto>
{
    private readonly I@(Model.EntityName)Repository _repository;

    public @(Model.EntityName)Service(I@(Model.EntityName)Repository repository)
    {
        _repository = repository;
    }

    public async Task<PageResult<@(Model.EntityName)Dto>> FindPageAsync(
        PageRequest<@(Model.EntityName)> request)
    {
        var query = _repository.Query()
            .WhereIf(!string.IsNullOrEmpty(request.Entity.@(Model.Columns[0].PropertyName)),
                e => e.@(Model.Columns[0].PropertyName).Contains(request.Entity.@(Model.Columns[0].PropertyName)));
        return await query.ToPageResultAsync(request);
    }
}
```

## 4. 表元数据结构

```csharp
public class TableMetaData
{
    public string TableName { get; set; }          // 物理表名: Sys_User
    public string EntityName { get; set; }         // 实体名: User
    public string ModuleName { get; set; }         // 模块名: Sys
    public string Namespace { get; set; }          // 命名空间
    public string TableComment { get; set; }       // 表注释
    public string PrimaryKey { get; set; }         // 主键列
    public List<ColumnMetaData> Columns { get; set; }
    public bool IsTreeTable { get; set; }          // 是否树表
    public bool IsMasterDetail { get; set; }       // 是否主子表
}

public class ColumnMetaData
{
    public string ColumnName { get; set; }         // 物理列名: user_code
    public string PropertyName { get; set; }       // 属性名: UserCode
    public string DbType { get; set; }             // 数据库类型: VARCHAR
    public string CSharpType { get; set; }         // C#类型: string
    public int? MaxLength { get; set; }
    public bool IsPrimaryKey { get; set; }
    public bool IsNullable { get; set; }
    public string Comment { get; set; }
    public bool IsBaseField { get; set; }          // 是否基类字段
    public bool IsQueryCondition { get; set; }     // 是否查询条件
    public string QueryType { get; set; }          // LIKE/EQ/BETWEEN
}
```

## 5. 模板类型

| 模板 | C# 后端 | Vue 前端 | 说明 |
|---|---|---|---|
| `crud` | Entity/Service/Controller/EF | List/Form/API | 单表 |
| `crud_tree` | TreeEntity/TreeService | TreeList/Form | 树表 |
| `crud_master` | Master+Detail Service | Master+Detail Table | 主子表 |
| `module` | 完整模块框架 (含csproj) | 无 | 快速创建模块 |
| `query` | 仅查询 Service | 查询页面 | 无编辑功能 |

## 6. 自定义模板机制

```csharp
public interface ICodeGenTemplate
{
    string TemplateName { get; }
    string Description { get; }
    /// <summary>根据元数据生成文件列表</summary>
    List<GeneratedFile> Generate(TableMetaData table);
}

public class GeneratedFile
{
    public string FileName { get; set; }           // UserService.cs
    public string RelativePath { get; set; }       // Application/Services/
    public string Content { get; set; }            // 文件内容
    public FileType Type { get; set; }             // CSharp/Vue/Sql
}
```
