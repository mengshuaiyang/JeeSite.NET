## 二、待建任务分类

### A. 新发现增强项（2026-06-12 扫描新增, 2 项)

#### [P2-1] UEditor 完整协议兼容 (扩展增强)
**来源**: JeeSite5 `common/ueditor` 12 类完整实现
**现状**: JeeSite.NET `EditorController` 仅提供基础上传/下载接口
**缺失协议子操作未实现**: `ActionMap` 15+ 子协议 (uploadimage/uploadscrawl/uploadvideo/uploadfile/catchimage/imageManager/listfile/listimage/crawlRemote/removeFile/config
**实现建议**:
- 实现 `UEditorActionHandler` 中间件，根据 `action` 参数路由至对应 Handler
- 实现 `ConfigManager` 等价配置读取 (JSON 模板化)
- 实现 `ImageHunter` 远程图片抓取 (防止外链图片本地化)
- 实现 `FileManager` 列表浏览 (按日期/目录浏览服务器文件)
**优先级**: P2 (仅在使用 UEditor JS 前端时需要，当前 Vue 3 自带编辑器已可替代)
**预估工作量**: 3-4 天
**是否阻塞项**: 无

#### [P2-2] Excel 自定义字段类型扩展点 (FieldType 体系)
**来源**: JeeSite5 `common/utils/excel/fieldtype` (BigDecimalType / MoneyType / FieldType 基类
**现状**: JeeSite.NET `ExcelFieldAttribute` 使用 Attribute 注解，无 `FieldType` 扩展体系
**功能缺口**: 无法通过继承扩展 `MoneyType` (金额格式) / `BigDecimalType` (精度控制) / `OfficeType` / `CompanyType` / `PostListType` / `RoleListType` / `AreaType` 等复杂类型
**实现建议**:
- 抽象 `IExcelFieldType` 接口 (GetFieldValue / SetFieldValue / Format)
- 内置 8 个类型实现 (decimal/money/office/company/post/role/area/datetime
- `ExcelService` 扫描属性上类型注册器 (Type Register Pattern)
**优先级**: P2 (当前注解方式已可满足 90% 的需求，仅复杂场景需扩展体系作为扩展
**预估工作量**: 2 天
**是否阻塞项**: 无

#### [P2-3] 短信/邮件验证码登录（login-by-valid-code）
**来源**: JeeSite5 LoginController + AccountController 支持手机验证码登录、邮箱验证码登录、验证码密码重置
**现状**: JeeSite.NET `AccountController` 的 `login-by-valid-code` 端点仅返回 "功能实现中"
**功能缺口**: 缺少短信发送接口、验证码存储校验、登录/注册/密码重置三种验证码场景、防暴力破解（限频/限流）
**实现建议**:
- `ValidCodeService.Generate(target, scene)` 生成验证码（6 位数字，5 分钟过期）
- `ValidCodeService.Verify(target, scene, code)` 校验并清理
- `SmsService.Send` 发送短信（集成阿里云 SMS/腾讯云 SMS）
- `EmailUtil.Send` 发送邮件验证码
- 限频：同一手机号/邮箱 1 分钟 1 次，10 分钟 5 次
**优先级**: P2
**预估工作量**: 2-3 天
**是否阻塞项**: 无

#### [P2-4] 菜单树/权限数据/路由信息端点（menuTree / menuRoute / authInfo）
**来源**: JeeSite5 LoginController 提供 menuTree / menuRoute / authInfo / index 4 个端点
**现状**: JeeSite.NET `useAppStore().loadMenus()` 仅加载菜单树，缺少按权限过滤、按角色聚合、路由元数据
**功能缺口**: 
- 缺少 `authInfo`：按用户聚合菜单和权限字符串（前端权限系统数据源）
- 缺少 `menuRoute`：路由表（component、path、meta.permission、meta.title）
- 缺少 `menuTree`：侧边栏菜单树（按权限过滤）
**实现建议**:
- `AuthService.GetAuthInfoAsync(userCode)` 返回菜单树 + 权限列表 + 用户信息
- `AuthService.GetMenuRouteAsync(userCode)` 返回 Vue Router 可消费的路由表
**优先级**: P2
**预估工作量**: 2 天
**是否阻塞项**: 无
