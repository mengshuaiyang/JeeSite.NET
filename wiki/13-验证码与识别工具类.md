# 工具类手册 · 验证码与识别工具

本章介绍图形验证码、User-Agent 解析、条形码 / 二维码生成与识别以及短信 / 邮件验证码发送相关工具。

---

## 一、CaptchaUtil

`CaptchaUtil`（`Utils/CaptchaUtil.cs`）生成包含随机文字或算术题的 PNG 图片，用于登录 / 注册 / 找回密码等场景的人机校验。

### 1.1 核心方法

**（1）生成随机文本图像**
```csharp
public static byte[] GenerateCaptchaImage(string text, int width = 120, int height = 40)
```
- 将 `text` 绘制到指定尺寸的 PNG 画布，返回图像字节数组。
- 内部会随机倾斜字体、添加噪点与干扰线，避免 OCR 识别。

**（2）生成随机文本**
```csharp
public static string GenerateRandomText(int length = 4)
```
- 从字符集 `0-9`、`A-Z`（排除 `0/O/1/I` 等易混字符）中随机抽取 `length` 个字符。
- 推荐 `length = 4`，兼顾识别率与安全性。

**（3）生成算术验证码**
```csharp
public static (string Expression, int Answer) GenerateMathExpression(int max = 20)
```
- 生成形如 `"3 + 7"`、`"12 - 5"`、`"4 × 6"` 的表达式。
- 返回 `(Expression, Answer)` 二元组，前端显示 `Expression`，后端用 `Answer` 做校验。

### 1.2 使用示例

```csharp
// 方案 A：字符验证码
string text = CaptchaUtil.GenerateRandomText(4);   // "A7K9"
byte[] png  = CaptchaUtil.GenerateCaptchaImage(text, 140, 44);
// 保存 Session / Redis：cache.Set("captcha:" + sessionId, text, TimeSpan.FromMinutes(2));
return File(png, "image/png");

// 方案 B：算术验证码
var (expr, ans) = CaptchaUtil.GenerateMathExpression(20);
byte[] png2 = CaptchaUtil.GenerateCaptchaImage(expr, 160, 44);
// cache.Set("captcha:" + sessionId, ans, TimeSpan.FromMinutes(2));
```

### 1.3 校验流程（ValidCodeController）

```csharp
// GET  /api/v1/sys/valid-code/captcha    → 返回图片
// POST /api/v1/sys/valid-code/captcha    → 校验用户输入
public async Task<IActionResult> VerifyCaptcha(string userInput)
{
    var expected = await _cache.GetAsync("captcha:" + sessionId);
    await _cache.RemoveAsync("captcha:" + sessionId);   // 一次性使用
    if (!string.Equals(expected, userInput, StringComparison.OrdinalIgnoreCase))
    {
        return BadRequest("验证码错误");
    }
    return Ok();
}
```

### 1.4 安全性建议

- **一次性**：验证通过后立即删除缓存，防止重放。
- **有效期**：通常不超过 2 分钟。
- **失败限流**：同一 IP 在 1 分钟内失败 5 次后强制冷却。
- **敏感操作**：转账 / 删除 / 重置密码 等场景优先使用短信 / 邮箱验证码。
- **高防场景**：可替换为滑动验证 / 行为验证（如 reCAPTCHA / hCaptcha / 极验）。

## 二、UserAgentUtil

`UserAgentUtil`（`Utils/UserAgentUtil.cs`）解析 HTTP `User-Agent` 字符串，提取浏览器 / 操作系统 / 设备类型。

### 2.1 核心数据结构

```csharp
public class UserAgentInfo
{
    public string Browser { get; set; }       // "Chrome 120"
    public string BrowserName { get; set; }   // "Chrome"
    public string BrowserVersion { get; set; }// "120.0"
    public string OS { get; set; }            // "Windows 11"
    public string DeviceType { get; set; }    // "Desktop" / "Mobile" / "Tablet"
    public bool IsMobile { get; set; }
    public bool IsBot { get; set; }           // 搜索引擎爬虫识别
    public string Raw { get; set; }           // 原始 UA 字符串
}
```

### 2.2 核心方法

```csharp
public static UserAgentInfo Parse(string userAgent)
public static string GetBrowser(string userAgent)
public static string GetOperatingSystem(string userAgent)
public static string GetDeviceType(string userAgent)
public static bool IsMobile(string userAgent)
```

### 2.3 支持的浏览器 / 系统

| 类别 | 识别项 |
|------|--------|
| **浏览器** | Chrome, Edge, Firefox, Safari, IE, Opera, QQ 浏览器, 微信, 支付宝 |
| **操作系统** | Windows 11 / 10 / 8 / 7, macOS, iOS, Android, Linux, HarmonyOS |
| **设备类型** | Desktop, Mobile, Tablet |

### 2.4 在 LogService 中的典型用法

```csharp
// Modules/Sys/Application/Services/LogService.cs
public async Task CreateAsync(LogDto dto)
{
    var ua = UserAgentUtil.Parse(HttpContext.Request.Headers["User-Agent"]);

    var entity = new Log
    {
        RequestPath = dto.RequestPath,
        UserId      = CurrentUser.Id,
        UserName    = CurrentUser.UserName,
        Browser     = ua.Browser,       // "Chrome 120"
        OS          = ua.OS,            // "Windows 11"
        DeviceType  = ua.DeviceType,    // "Desktop"
        IP          = HttpContext.GetRemoteIpAddress(),
        // ...
    };
    await _repository.InsertAsync(entity);
}
```

## 三、BarcodeUtil

`BarcodeUtil`（`Utils/BarcodeUtil.cs`）基于 **ZXing.NET** 生成和识别条形码、二维码。

### 3.1 依赖

```xml
<PackageReference Include="ZXing.Net" Version="0.16.9" />
<PackageReference Include="ZXing.Net.Bindings.ImageSharp" Version="0.16.9" />
```

### 3.2 核心方法

**（1）生成二维码**
```csharp
public static byte[] GenerateQrCode(string text, int size = 256)
```
- 默认 QR Code Version 自动，纠错级别 `M`（约 15% 可恢复）。
- 返回 PNG 字节数组。

**（2）生成带 Logo 的二维码**
```csharp
public static byte[] GenerateQrCodeWithLogo(
    string text,
    int size = 256,
    string logoPath = null,
    int logoSizeRatio = 5)        // logo 占整张二维码 1/5
```
- 将 `logoPath` 图像（PNG 透明最佳）叠加到二维码正中。
- 中心区域由于纠错能力存在，小比例 Logo 不影响识别。

**（3）生成 Code128 条形码**
```csharp
public static byte[] GenerateBarcode(string text, int width = 300, int height = 80)
```
- 使用 Code128 编码（支持全 ASCII）。

**（4）从图片识别内容**
```csharp
public static string ReadBarcode(Stream imageStream)
public static string[] ReadBarcodes(Stream imageStream)
```
- 识别图像中的一个 / 多个条形码或二维码内容。

### 3.3 使用示例

```csharp
// 生成二维码
byte[] qr = BarcodeUtil.GenerateQrCode("https://your-site.com/article/123", size: 256);
return File(qr, "image/png");

// 生成带 Logo 的二维码（如公司 Logo）
byte[] qrWithLogo = BarcodeUtil.GenerateQrCodeWithLogo(
    "https://your-site.com",
    size: 400,
    logoPath: "wwwroot/logo.png");

// 生成条形码
byte[] bc = BarcodeUtil.GenerateBarcode("SKU-20250001", width: 300, height: 80);

// 从上传图片识别
using var stream = Request.Form.Files[0].OpenReadStream();
string content = BarcodeUtil.ReadBarcode(stream);
```

## 四、ValidCodeService

`ValidCodeService`（`Modules/Sys/Application/Services/ValidCodeService.cs`）负责短信 / 邮件验证码的生成、发送与校验。它与上面 `CaptchaUtil` 的区别：

| 维度 | CaptchaUtil（图形） | ValidCodeService（短信 / 邮件） |
|------|---------------------|-------------------------------|
| 交互对象 | 用户 → 服务器 | 用户 → 手机 / 邮箱 → 服务器 |
| 用途 | 人机区分 | 身份核验（绑定手机 / 找回密码 / 登录） |
| 生命周期 | 1 ~ 2 分钟，一次性 | 5 分钟，一次性 |
| 失败风险 | 自动识别（OCR） | 暴力破解 → 需要限流 |

### 4.1 核心方法

```csharp
// 生成并发送短信验证码
public async Task<SendResult> SendSmsAsync(string mobile, string purpose = "login")

// 生成并发送邮件验证码
public async Task<SendResult> SendEmailAsync(string email, string purpose = "login")

// 校验验证码
public async Task<bool> VerifyAsync(string key, string code)
```

### 4.2 限流策略

- 同一手机号 **1 分钟最多 1 次**。
- 同一手机号 **10 分钟最多 5 次**。
- 同一手机号 **24 小时最多 20 次**。
- 验证码校验失败 5 次，冻结 15 分钟。

```csharp
// 示例限流代码
string rateKey = $"validcode:rate:{mobile}";
if (await _cache.GetAsync(rateKey) != null)
{
    return SendResult.Fail("发送过于频繁，请稍后再试");
}
await _cache.SetAsync(rateKey, "1", TimeSpan.FromMinutes(1));
```

### 4.3 缓存结构

```text
Key       : validcode:{purpose}:{mobile or email}
Value     : "123456"（6 位数字）
TTL       : 5 分钟

Key       : validcode:rate:{mobile or email}
Value     : 1（仅作为存在性标记）
TTL       : 1 分钟 / 10 分钟
```

### 4.4 示例：找回密码

```csharp
// 1. 发送验证码
var res = await _validCodeService.SendSmsAsync(mobile, "reset-password");
if (!res.Success) return BadRequest(res.ErrorMessage);

// 2. 校验验证码 + 重置密码
if (!await _validCodeService.VerifyAsync($"reset-password:{mobile}", inputCode))
{
    return BadRequest("验证码错误或已过期");
}
await _userService.ResetPasswordAsync(mobile, newPassword);
```

### 4.5 典型控制器端点（ValidCodeController）

```text
GET  /api/v1/sys/valid-code/captcha                  → 图形验证码图片
POST /api/v1/sys/valid-code/verify-captcha           → 校验图形验证码
POST /api/v1/sys/valid-code/send-sms                 → 发送短信验证码
POST /api/v1/sys/valid-code/send-email               → 发送邮件验证码
POST /api/v1/sys/valid-code/verify                   → 校验短信 / 邮件验证码
```
